# Rule-Based AI Chatbot

## Project Description

This project is a simple Rule-Based AI Chatbot developed using Python.

The chatbot responds to predefined user inputs using dictionaries and conditional logic.

## Features

- Handles greetings
- Responds to common questions
- Uses Python dictionaries
- Runs continuously using a loop
- Supports exit command

## Technologies Used

- Python 3

## How To Run

```bash
python chatbot.py
```

## Example

User: hello

Bot: Hi there!

User: how are you

Bot: I am doing great!
